3. Explain the difference between RAM and ROM.
	- RAM (Random Access Memory) isa volatile memory used by the computer to store data temporarily while programs are running
	- it allows for quick read and write access enabling smooth performance during tasks
	- when computer is powered off any data stored in RAM is lost
	- ROM (Read-Only Memory) is non-volatile memory that stores firmware and essential instructions required for the computers boot process
	- the data in ROM stays intact even when the power is removed
	- RAM is crucial for active processes while ROM for system startup and basic functionality

7.  What steps would you take to troubleshoot a network connectivity issue?
	- to troubleshoot a network connectivity issue, first , verify physical connections by checking cables, switches and routers
	- next confirm that device has a valid ip address like using command like ipconfig on Windows or ifconfig on Linux
	- Check if other device can connect to to the network
	- restarting the router or modem can resolve temporary glitches
	- if issues persist, ping the router and external websites to asses connectivity
	- lastly, review firewall settings and ensure that the appropriate ports are open as they may be blocking access
